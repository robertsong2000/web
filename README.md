# 开源项目列表网页

这是一个展示各类优秀开源项目的静态网页，旨在帮助开发者快速发现和了解不同领域的开源项目。

## 功能特性

- 分类展示：项目按类别（前端、AI、嵌入式、工具）进行分类
- 简洁界面：清晰的项目名称和简短描述
- 快速访问：点击项目名称即可跳转到项目主页

## 使用方法

1. 克隆本仓库
2. 打开`index.html`文件即可查看

## 贡献指南

欢迎提交新的开源项目！请按照以下步骤操作：

1. Fork 本仓库
2. 在`index.html`中添加新项目
3. 提交 Pull Request
